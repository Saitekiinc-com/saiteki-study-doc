# 進捗管理シート

このシートは、研修生とメンターが共有し、学習の進捗状況を可視化するためのものです。

## 1. スキルチェックリスト

### 基礎編
- [ ] HTMLの基本構造を理解している
- [ ] CSS Flexbox/Grid を使ってレイアウトが組める
- [ ] ES6+ の主要な構文（アロー関数、分割代入など）を使える
- [ ] TypeScript の基本型と型推論を理解している
- [ ] React コンポーネントを作成し、Propsでデータを受け渡せる
- [ ] `useState` を使って状態管理ができる
- [ ] `useEffect` の基本的な使い方と依存配列の意味を理解している

### 実践編
- [ ] プロジェクトのディレクトリ構成を理解している
- [ ] コンポーネントを適切に分割できる
- [ ] APIからデータを取得して表示できる
- [ ] ルーティングを設定できる
- [ ] コードのフォーマットとLintエラーを修正できる
