# 📚 JavaScript Primer 迷わないための入門書

## 概要
*   **書籍名**: JavaScript Primer 迷わないための入門書
*   **著者**: azu, Suguru Inatomi
*   **リンク**: https://jsprimer.net/

## レビュー

### 👍 Positive (良かった点・学び)
*   **非同期処理の解説が神**: `Promise` や `async/await` だけでなく、その裏側にある「イベントループ」の仕組み図解が非常にわかりやすかった。これまで「なんとなく」使っていた非同期処理の挙動が、自信を持って説明できるようになった。
*   **ES2015以降がベース**: 古い `var` や `prototype` の話から入るのではなく、現代的な `const/let` や `class` から学べるので、実務ですぐに使える知識が身についた。
*   **無料**: Webで全編無料で読めるのがすごい。

### 👎 Negative (難しかった点・合わなかった点)
*   **ボリューム**: 第一部だけでかなりの量があり、通読するのは骨が折れる。最初は「非同期処理」や「オブジェクト」など、つまづいている章だけ読むのが良いかもしれない。
*   **DOM操作**: ブラウザAPI（DOM操作）の記述は少なめなので、UIを作りたい人は別の本も併用する必要がある。
